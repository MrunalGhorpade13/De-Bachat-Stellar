## 2. Export to Excel
1. In your Google Form, go to the **"Responses"** tab.
2. Click the green icon **"Link to Sheets"**.
3. Create a new sheet named "De-Bachat Feedback Responses".
4. When done, go to **File > Download > Microsoft Excel (.xlsx)**.

## 3. Link to README
Once you have the Excel file, upload it to your GitHub repository and link it in the root `README.md`.
